# NOTE APP 
### Requirements
- create new note
- Add note summary titles
- Sync notes with localStorage
- delete note
- show more notes
- Move modified notes to the top of the list

### The challenge
Users should be able to:
- View the list of notes created.
- create new note 
- show more notes
- Sync notes to localstorage
